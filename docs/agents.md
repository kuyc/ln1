1. Назначение документа

Этот документ жёстко определяет роли, обязанности и ограничения всех агентов и нейросетей, участвующих в проекте ln1.

Документ обязателен для:

всех нейросетей;

всех агентов;

всех разработчиков.

Если агент действует вне рамок этого документа —
➡️ его действия считаются ошибочными и подлежат отклонению.

2. Главный принцип работы агентов

Агент — исполнитель.
Документация — закон.
Архив — источник истины.

3. Источник истины (КРИТИЧЕСКОЕ ПРАВИЛО)
3.1 Архив-first подход

Единственный источник истины проекта ln1 —
ПОСЛЕДНИЙ архив проекта, загруженный пользователем в память проекта ChatGPT.

Все агенты и нейросети ОБЯЗАНЫ:

Перед любым ответом считать последний архив актуальным.

Работать только с файлами из архива.

Игнорировать старые сообщения и предыдущие версии.

Если архив не загружен или устарел —
➡️ агент обязан остановиться и запросить архив.

3.2 Запрещённое поведение

Агенту ЗАПРЕЩЕНО:

опираться на память;

«додумывать» файлы;

смешивать версии проекта;

продолжать работу без архива.

4. Рабочая область проекта

Единственная допустимая рабочая область:

ln1/
 ├── docs/
 ├── src/
 ├── tests/
 ├── README.md
 └── .env.example


Файлы вне ln1/ не считаются частью проекта.

5. Управляющие роли
5.1 Product Manager

Ответственность:

продуктовая логика;

режимы работы;

приоритеты;

развитие проекта.

Запрещено:

писать код;

менять архитектуру напрямую.

5.2 Context Manager (КРИТИЧЕСКИЙ)

Ответственность:

контроль соответствия действий документации;

выявление рассинхронизации;

требование остановки работы.

Права:

блокировать продолжение;

требовать новый архив;

указывать на нарушения правил.

5.3 Agent Organizer

Ответственность:

распределение задач между агентами;

контроль границ ролей;

предотвращение конфликтов.

12. Ответственность за нарушения

Если агент:

игнорирует архив;

действует вне docs;

нарушает свою роль;

➡️ его результат не принимается.

13. Финальное правило агентов

Если нет архива — агент молчит.
Если нет документации — агент не действует.
Контроль важнее автономности.
